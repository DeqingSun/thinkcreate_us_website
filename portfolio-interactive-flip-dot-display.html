Interactive Flip-Dot Display

A kinetic installation that displays real-time hand drawings, videos, still images and text, or interacts with people.
· Interactive Installation    · Kinetic Installation   
 Previous Back to all works  Next

The Interactive Flip-Dot Display is a kinetic art installation. It has three modes:

1. Video Mode: display video files, static images and text.

2. Interactive Mode: real-time body tracking and display with the included 3D depth camera.

3. Drawing Mode: real-time display drawings from the included touch screen.

The size of the installation is customizable. Because of its modular design, the installation can be as small as a desktop art gadget made of a single unit, or as big as a wall made of hundreds of units.

The color of dots on the display is customizable as well.
